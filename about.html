Who I am.

I'm a full-stack web developer based in California, working at HDT Technologies where I help plan and build production-grade web projects across the full stack. Outside of that, I independently build and manage websites for local businesses — plumbers, salons, auto shops, groomers, and more — giving small businesses a real presence online without the bloat or the big agency price tag.

Every site I ship is custom-designed, fast-loading, and backed by GitHub version control with automatic Render deployment. I also build the supporting infrastructure: client portals for self-service edits, Google Analytics dashboards so owners can see who's actually visiting, and Discord alerts when changes go live.

I care a lot about the craft — writing clean code, designing things that feel premium, and making sure the people using what I build actually enjoy it. If that sounds like something your business needs, let's talk.
